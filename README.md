Logistic-Regression
============================

A module to use Logistic Regression in OpenCV using <b>C++</b> API to perform <b>2 class</b> or <b>multi-class classification</b>
INSTALLATION PROCEDURE:
============================
cd <path to LogisticRegression-cpp>
<br/>
mkdir build
<br/>
cd build
<br/>
cmake ..
<br/>
make
<br/>
./LR
